# 欢迎使用 Next.js 博客模板

这是一篇使用 MDX 编写的示例文章。

## 功能特点

- ✅ Next.js 14 + App Router
- ✅ TypeScript
- ✅ Tailwind CSS
- ✅ MDX 内容支持

## 代码示例

```typescript
const greeting = (name: string) => {
  return `Hello, ${name}!`
}

console.log(greeting('World'))
```

## 开始使用

1. 点击 "Use this template" 创建新博客
2. 在 `content/posts/` 目录添加文章
3. 开始写作吧！
